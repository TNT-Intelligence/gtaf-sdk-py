# Security Policy

## Supported Versions

The following versions of gtaf-sdk are currently supported with security updates:

| Version | Supported |
|---------|------------|
| 0.1.x   | ✅         |
| < 0.1.0 | ❌         |

Security updates are provided only for the latest minor release line.

---

## Scope

gtaf-sdk is an optional integration helper layer built on top of gtaf-runtime.

Normative enforcement and projection semantics are defined exclusively by gtaf-runtime.

Security issues affecting enforcement logic must be reported against gtaf-runtime.

---

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately via:

contact@tnt-intelligence.com

Please include:

- A clear description of the issue
- Steps to reproduce
- Affected version
- Expected vs. observed behavior

Do not disclose security vulnerabilities publicly before coordinated disclosure.

You can expect an acknowledgment within 3 business days.
